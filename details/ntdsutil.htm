- ツール概要

カテゴリ
情報収集
説明
Active Directory データベースの保守に使用する。
攻撃時における想定利用例
NTDS.DIT(NTDSのデータベース)を抽出し、他のツールを用いてパスワードを解析する。

- 実行成功時に確認できる痕跡

  • ntdsutil.exeが実行され、イベントログ「セキュリティ」にイベントID: 8222が記録されている
  • オブジェクト "[システムドライブ]\SNAP_[日時]_VOLUME[ドライブレター]$" に対するハンドルの要求が成功している
